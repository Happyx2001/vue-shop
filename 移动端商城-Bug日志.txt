购物街项目
Bug.1：
Swiper 轮播图运行不了，点击轮播图图片链接进去之后再返回就又能运行了。很迷惑

解决方案：网络因素，Swiper源码中，定时器设置的开启时间过快，导致图片还未加载完成轮播图代码就执行完毕。导致轮播图代码识别不到图片，因此不会有轮播图效果。

Bug2：
Better-scroll的Bug合集，刷新不过来导致scroll高度无法及时刷新，scroll无法滚动

Bug3：
箭头函数缩写产生的问题，当箭头函数的返回值有一行代码时，可以不用写 return 
例如：
(item) => {        // 带有花括号时，不加return 会没有返回值
return item++
}
简写为：           // 简写默认返回值为 一行代码的结果
item => item++

项目简介：基于Vue开发的一个移动端购物商城
技术栈：Vue、Vue-router、Vuex、axios
项目技术细节：

使用px2rem-loader插件转换px为rem，实现了项目移动端的适配
使用Vue-lazyload实现了图片懒加载，优化了一定的性能
使用Vuex实现了添加商品至购物车的功能
封装了axios，便于项目不同区域请求api接收数据
封装了项目的导航栏、标签栏、返回顶部等组件，实现了项目的组件化开发
